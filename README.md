# Walmart Sales Analysis

## Project Overview
This project analyzes Walmart weekly sales data using SQL, Excel, Power BI, statistics, and Lean Six Sigma framing to identify key revenue-driving stores, evaluate sales variability, and generate actionable business insights.

## Dataset
- 7,625 weekly sales records
- 45 stores
- Date range: February 2010 to October 2012

## Tools Used
- SQL
- Excel
- Power BI
- Statistics
- Lean / Six Sigma

## Analysis Performed
- Data cleaning and validation
- KPI analysis
- Store ranking
- Pareto analysis
- Variability and risk analysis
- Correlation analysis
- Hypothesis testing
- Confidence interval estimation
- Dashboard creation

## Key Findings
- Top 26 stores contributed about 80.44% of total sales
- 14 stores were classified as high-risk based on variability
- Holiday weeks showed significantly higher average sales than non-holiday weeks
- External factors showed weak correlation with sales

## Statistical Validation
- Holiday mean sales: 1,154,744.93
- Non-holiday mean sales: 1,068,098.15
- p-value: 0.00275
- 95% confidence interval: (30,044.93, 143,248.63)
